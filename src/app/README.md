# `src/app` — legado/inativo

Este diretório **não é a árvore oficial ativa** do sistema.

## Status atual
- A árvore principal usada pelo projeto está em `app/`.
- O núcleo funcional atual está em:
  - `/`
  - `/login`
  - `/rebuild/admin`
  - `/rebuild/operator`
- `src/app/gerencia` foi mantido apenas como referência histórica.

## Regra para novos desenvolvimentos
Não adicionar novas rotas ativas aqui.
Use `app/` para qualquer página/layout/rota oficial do sistema.
